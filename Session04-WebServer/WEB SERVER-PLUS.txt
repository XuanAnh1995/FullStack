* CHẾ ĐỘ CÀI GLOBAL --global -g
* CHỈ BIẾN MẤT KHI MÌNH CHỦ ĐỘNG GỠ, CÀI LẠI WINDOWS
* CÀI XONG NẰM Ở : 	C:\USER\<login name>\APPDATA\ROAMING\NPM\NODE_MODULE

	npm install --global vite		npm install --global serve
	npm install -g vite			npm install -g serve
	npm i --global vite			npm i --global serve
	npm i -g vite				npm i -g serve

	npm r -g vite				npm r -g serve
